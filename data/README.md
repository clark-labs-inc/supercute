# Data directory

This directory contains generated benchmark artifacts and supplied experiment outputs.

| File | Description |
|---|---|
| `benchmark.jsonl` | Canonical harvested benchmark examples from the original SUPERCUTE run. |
| `hard.jsonl` | Accumulated model-failure examples used to build the benchmark. |
| `audit_hard.jsonl` | Separate document-audit track examples. |
| `eval_results.jsonl` | Raw per-call six-model results for the paper. |
| `eval_summary.json` | Wilson-interval summary generated by `python -m supercute.analyze_eval`. |
| `benchmark_3model_results.txt` | Earlier three-model mixed-benchmark result table. |
| `finqa_records.jsonl`, `tabfact_records.jsonl`, `sec_records.jsonl` | Derived records fetched by the included public-data scripts in the original project. |

For a strict redistribution release, verify the license of every public source used to create derived records. PublicLift itself does not require redistributing third-party rows; it provides generators and adapter contracts.
